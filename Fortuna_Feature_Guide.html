The CBM Mantra — Every page in Fortuna enforces three principles:
1. Design for the Category — CEPs describe category buying occasions, not brand-specific moments.
2. Analyse for the Buyer — Non-buyers and light buyers are always the primary analysis segment.
3. Report for the Brand — All scores are Double Jeopardy normalised. Deviation from expected score matters, not the raw number.

03

Dashboard Overview

http://localhost:3000/dashboard
Overview

The command centre. CEO-ready summary of the current wave with segment-aware KPIs, strategic insight, and gap diagnosis. All metrics update dynamically when the segment toggle changes.

How to Use
  • Segment Toggle (top-right) — switch between Non-buyers + Light (default), Heavy Buyers, Overall.
  • KPI Cards — MPen, MMS, NS, SoM with wave-on-wave delta (green triangle up = improved).
  • Key Insight Panel — read the headline. Three metric bars explain the drivers. Bottom recommendation = single next action.
  • MMS vs SMS Gap bar — identifies whether the gap is mental or physical availability.
  • Click Add Data to open the data entry modal.
How to Interpret Data and Results
  • Non-buyers + Light is always the primary segment. This is the growth pool. Low MPen here = limited mental availability among non-buyers.
  • MMS greater than SMS gap = Physical availability problem. Buyers think of the brand but cannot buy it. Fix distribution first.
  • SMS greater than MMS gap = Mental availability problem. Buyers can find the brand but do not think of it first. Fix CEP reach in media.
  • A +2pp MPen increase in non-light segment is a significant growth signal — more non-buyers are linking the brand to at least one buying occasion.
  • Heavy Buyers segment: high SoM is expected. Concern is if NS is low — loyal buyers associate the brand with few occasions.
04

Brand KPIs (Analytics)

http://localhost:3000/dashboard/analytics
Overview

Competitive benchmarking page. All brands shown side-by-side, DJ-normalized for fair comparison across different-sized brands. Focal brand highlighted in neon green. Rankings shift per segment.

How to Use
  • Chart ranks brands by the active segment primary KPI — MPen for non-light, SoM for heavy, MMS for overall.
  • Change the segment toggle to see how competitive rankings shift between buyer groups.
  • Competitive Analysis panel below the chart gives a strategic summary for the current segment.
  • Data Requirements checklist shows what is missing — red items must be resolved before reporting.
How to Interpret Data and Results
  • Double Jeopardy check: Brands with higher MPen should also have higher NS. High MPen + low NS = buyers know you broadly but you are not used across many occasions.
  • MMS vs SoM: MMS greater than SoM = outperforming mentally, expect sales growth. SoM greater than MMS = living off past equity, mental decline coming.
  • NS is the strongest quality indicator. MPen 40% NS 4.5 outperforms MPen 55% NS 2.1 in long-term growth potential.
  • Look for competitors with high MPen but low MMS — they spread thin. A targeted CEP campaign can steal mind share from them.
05

Mental Advantage Map

http://localhost:3000/dashboard/mental-advantage
Overview

Brand x CEP heatmap showing performance relative to DJ-expected score. The most strategically actionable page — directly drives creative briefs and media planning decisions.

How to Use
  • Each cell shows deviation from DJ-expected score (Actual minus Expected, in percentage points).
  • Green cells (DEFEND, at least +5pp) — mental assets. Feature consistently in all media.
  • Gray cells (MAINTAIN, within 4pp) — on-par. Hold current activity.
  • Red cells (BUILD, minus 5pp or worse) — mental gaps. Prioritise in next campaign brief.
  • Strategic Summary cards count DEFEND / MAINTAIN / BUILD cells per brand.
How to Interpret Data and Results
  • DJ Expected Score: Expected = (Row Total x Column Total) divided by Grand Total. Removes the size effect — large brands naturally get more linkages.
  • DEFEND cells are media assets. A +17pp deviation means the brand wins disproportionately there — anchor all media to it.
  • BUILD cells are opportunities. A -14pp deviation means one targeted campaign can shift this significantly.
  • Read down columns (CEP view): All MAINTAIN cells in a column = no brand owns that occasion — blue ocean opportunity.
  • Urgent signal: If a competitor holds DEFEND on the most important category CEP and you hold BUILD, that is the highest-priority strategic gap.
06

Physical Availability

http://localhost:3000/dashboard/physical
Overview

Three-pillar physical availability audit: Presence (where buyers find you), Portfolio (what they buy), and Prominence (visibility at point of purchase). Five tabs cover the full physical picture.

How to Use
  • Overview Tab — four KPI cards give headline numbers. Three pillar cards explain each score.
  • Presence Tab — channel coverage vs category norm. Shows if coverage is above or below category average per channel.
  • Portfolio Tab — SKU Efficiency Ratio (Sales% divided by Penetration%). Above 1.0 = star product. Below 1.0 = underperforming.
  • Prominence Tab — Fame x Uniqueness matrix. Top-right quadrant = Distinctive Brand Assets (DBAs).
  • Gap Analysis Tab — changes per segment toggle. Shows MMS vs SMS gap and physical actions to address it.
How to Interpret Data and Results
  • Coverage below category norm in high-traffic channels = direct explanation for MMS vs SMS gap. Fix distribution before increasing media spend.
  • Efficiency Ratio above 1.5: Disproportionate sales driver — always in stock, prominently placed.
  • Efficiency Ratio below 0.5: SKU has reach but does not convert — investigate pricing, placement, or relevance.
  • High fame / low uniqueness asset = generic. Buyers know it but it does not distinguish the brand. Refresh.
  • High uniqueness / low fame asset = hidden gem — invest in amplification.
07

Marketing Reach

http://localhost:3000/dashboard/reach
Overview

Measures advertising effectiveness: whether buyers noticed the creative (Effective Reach) and whether they correctly attributed it to the brand (Correct Branding). Uses de-branded stimuli for unbiased measurement.

How to Use
  • Four summary KPI cards give the overall picture for the current wave.
  • Trend chart — Noticed % (dark) vs Correctly Branded % (green) per creative across waves.
  • Creative Performance Table — all creatives with scores and rating: Strong / Medium / Weak.
  • Branding Ratio (Branded divided by Noticed) = % of people who saw the ad and knew it was yours. Most important single metric.
How to Interpret Data and Results
  • Branding Ratio below 60% = Vampire Creative. Noticed but not attributed to brand. Media spend builds category awareness, not brand awareness.
  • Branding Ratio above 80% = Strong branding. Target for all creatives.
  • High Noticed% + Low Branded% = attention-grabbing but brand-generic. Increase logo, brand colours, distinctive asset presence.
  • Low Noticed% + High Branded% = well-branded but not reaching enough people. Media placement or frequency problem, not creative.
  • Cross-reference with Mental Advantage Map: if a campaign aimed to build a specific CEP but Branding Ratio is below 60%, the spend went to category, not brand.
08

WOM Tracker

http://localhost:3000/dashboard/wom
Overview

Tracks Positive Word of Mouth (PWOM) and Negative Word of Mouth (NWOM) per 100 category buyers. Binary past-month measurement — not NPS. WOM is one of the strongest organic growth levers in CBM.

How to Use
  • Four summary cards — PWOM, NWOM, Net WOM (PWOM minus NWOM), WOM Ratio (PWOM divided by NWOM).
  • Competitive WOM Table — all brands ranked by Net WOM. Single number to track for organic growth health.
  • Trend Chart — PWOM and NWOM wave-on-wave for the focal brand.
How to Interpret Data and Results
  • Net WOM negative = more people warning against than advocating. Manage customer experience before increasing media spend.
  • WOM Ratio above 4x = strong. For every person warning against, five are recommending. Healthy organic engine.
  • NWOM spike (sudden increase) often precedes brand health decline by 1-2 waves. The earliest warning signal in the platform.
  • High PWOM + High Reach = compound growth (earned and paid amplify each other).
  • PWOM greater than MPen = people recommending without strong CEP link. Brand inertia — will erode without investment.
09

Forecast Engine

http://localhost:3000/dashboard/forecast
Overview

Five forecast tools: Double Jeopardy Law, CEP expected score matrix, MMS-to-Sales correlation, AI media simulation, and growth potential waterfall. Use to build the investment case and set next-wave targets.

How to Use
  • DJ Forecast tab: Scatter of brands on DJ curve + scenario builder. Set target MPen to predict resulting NS.
  • CEP Expected Scores tab: Full brand x CEP matrix with Actual / Expected / Deviation. Export to Excel.
  • MMS to Sales tab: Dual-line chart with 1-2 quarter lead projection (r=0.83 correlation).
  • AI Predictions tab: Media spend multiplier slider, category twin benchmarks, AI trend alerts.
  • Growth Potential tab: MPen by segment, growth waterfall, Recruitment vs Network Size priority.
How to Interpret Data and Results
  • Brands above the DJ curve have disproportionately loyal buyers. Brands below have distribution or relevance problems.
  • MMS rising but SMS flat = expect SMS to follow in 1-2 quarters. This is the invest-now, harvest-later argument.
  • Growth Potential bars above zero = untapped MPen among non-buyers who know the category. Easiest growth to unlock.
  • Recruitment priority (low MPen, acceptable NS): invest in reach — more non-buyer exposure.
  • NS priority (acceptable MPen, low NS): invest in CEP breadth — more occasion associations.
10

CEP Builder

http://localhost:3000/dashboard/ceps
Overview

Design the attribute list for your brand tracking survey using the 7W framework (Who, What, When, Where, Why, With What, With Whom). CBM Engine validates every attribute in real time and enforces correct portfolio composition.

How to Use
  • Use 7W tabs to organise CEPs by buying occasion dimension — ensures balanced coverage.
  • Classify each attribute as CEP, Baseline, or Secondary.
  • Composition Meter — target 60-70% CEPs, max 30% Baseline, max 20% Secondary.
  • Attributes with superlatives (most, best, more, least, worst, first, only) are flagged — rephrase in plain language.
  • Click AI Generate to auto-draft a portfolio. Use Critique mode to audit an existing list.
How to Interpret Data and Results
  • Wording rule: "Accepted everywhere I shop" is correct. "The most accepted brand" is a violation (superlative).
  • Lists dominated by WHAT and WHY miss WHEN and WHERE — the highest-traffic entry points in most categories.
  • Composition red at above 30% Baseline = functional attributes dilute the occasion-based signal.
  • AI Generate: review every suggestion — AI applies CBM rules but does not know your market's cultural nuance.
11

Surveys

http://localhost:3000/dashboard/surveys
Overview

Build and manage fieldwork waves. Block-based survey canvas with 17-block library, CBM compliance checking, Excel upload pipeline, and full distribution system for email and WhatsApp.

How to Use
  • Survey Waves tab — all waves with status, response progress, and compliance score.
  • 17-Block Library tab — all CBM question blocks. Core blocks (CORE badge) are mandatory.
  • Survey Builder tab — drag blocks onto canvas. Core blocks cannot be removed.
  • Upload Results tab — drag and drop Excel fieldwork file. System validates, calculates KPIs, shows preview.
  • Distribute tab — send invitations via Email (Resend/SMTP) or WhatsApp (Meta Cloud API). Paste or upload CSV of recipients.
How to Interpret Data and Results
  • Compliance score below 100% = expand compliance panel to see which CBM rules are failing.
  • Core blocks must never be removed: Category Screener sets the sampling frame. Mental Availability (block 09) is the most valuable data.
  • Email suits professional samples. WhatsApp is more effective for consumer panels in mobile-first markets like Indonesia.
  • CSV format: headers must be "name", "email", "phone". Phone in E.164 format (+62... for Indonesia).

Reports

http://localhost:3000/dashboard/reports
Overview

Generate client-ready outputs from the latest wave data. Download a branded 6-slide PPTX deck or PDF executive summary — both auto-populated from live DB data, with demo data fallback.

How to Use
  • Select the wave and segment from the dropdowns.
  • Click Download PPTX — generates: Cover, KPI Scorecard, Mental Advantage Map, WOM, Marketing Reach, Methodology.
  • Click Download PDF — generates: cover, KPI table, mental advantage summary, WOM table, footnotes.
  • API access: /api/reports?waveId=ID&format=pptx&segment=NON_LIGHT
How to Interpret Data and Results
  • PPTX Slide 2 (KPI Scorecard): focal brand row in green. Any brand with MMS greater than SoM has untapped sales potential.
  • PPTX Slide 3 (Mental Advantage Map): DEFEND cells = strengths to present. BUILD cells = priorities for next planning cycle.
  • PPTX Slide 5 (Marketing Reach): Branding Ratio below 60% = flag creative for rework in client meeting.
  • PDF Executive Summary: MMS vs SMS gap is the single most important number for the CEO conversation.

Settings — Integrations and API

http://localhost:3000/dashboard/settings
Overview

Platform configuration hub. The Integrations and API tab (default) connects all seven external services in one place. Additional tabs cover appearance, layout, regional, and data/privacy controls.

How to Use
  • Connection Status panel — shows at a glance which services are connected (Core, AI, Distribution).
  • Click any integration card to expand. Enter API keys or service URLs.
  • Secret fields have a show/hide toggle (eye icon). Use Test connection to verify before saving.
  • .env.local template at the bottom — click Copy All to get a ready-to-fill template.
  • Setup guide links open the provider documentation in a new tab.
How to Interpret Data and Results
  • Demo Mode (amber) = unconfigured but app still functional with sample data.
  • Not configured (red) = feature unavailable. Without ANTHROPIC_API_KEY, AI CEP Generator returns demo CEPs only.
  • Priority setup order: 1) DATABASE_URL, 2) Clerk auth, 3) Analytics Service, 4) Anthropic Claude, 5) Resend or SMTP, 6) WhatsApp Meta API.
  • WhatsApp webhook URL must be registered in Meta App Dashboard before delivery receipts will work.
